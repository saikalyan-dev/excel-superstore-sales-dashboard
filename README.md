# 📊 Superstore Sales Dashboard (Microsoft Excel)

## 📌 Project Overview

This project demonstrates the development of an interactive Sales Dashboard in Microsoft Excel using the Superstore dataset. The dashboard provides a concise overview of business performance through KPIs, Pivot Tables, Pivot Charts, and interactive slicers.

---

## 📈 Key Performance Indicators

- 💰 Total Sales: **$2,297,200.86**
- 📊 Total Profit: **$286,397.02**
- 📦 Total Orders: **9,994**
- 📈 Profit Margin: **12.47%**

---

## 📊 Dashboard Features

- KPI Cards
- Profit by Category
- Profit by Region
- Yearly Sales Analysis
- Monthly Sales Analysis
- Top 10 Customers by Profit
- Top 10 Products by Profit
- Interactive Slicers

---

## 🛠️ Excel Skills Used

- Pivot Tables
- Pivot Charts
- Dashboard Design
- Data Visualization
- IF
- COUNTIF
- SUMIF
- SUMIFS
- VLOOKUP

---

## 📌 Business Insights

- Technology generated the highest profit among all product categories.
- The West region contributed the highest profit.
- Sales show an upward trend across the years.
- Monthly sales analysis highlights seasonal performance.
- Top-performing customers and products were identified based on profit.
